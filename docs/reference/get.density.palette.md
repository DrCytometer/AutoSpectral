# Get Density Color Palette

Returns a good color palette for plotting cell density distribution.

## Usage

``` r
get.density.palette(dens, asp)
```

## Arguments

- dens:

  A numeric vector representing the density distribution.

- asp:

  The AutoSpectral parameter list. Prepare using
  `get.autospectral.param`

## Value

A vector of colors representing the density palette.
